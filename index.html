These are some tips for the lists in html:

  • <ul> is used when order is not important
  • <ol> is used when order is important. For example septs by steps
  • you can use <ul> or <ol> inside a list item <li></li>. Like now:
    • item1
    • item2
  • You can use css attributes to change the style of the list. Like:
    • list-style-type
    • list-style-image
    • list-style-position
  • these are some examples from the types of numbers and bullets:
    • Bullets for <ul>
      • disc
      • circle
      • square
    • Numbers for <ol>
      1. decimal
      2. lowee-latin
      3. lower-roman
